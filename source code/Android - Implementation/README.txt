This is a program for implementation of Real Time Text Recognition.

To run this program you can install "robotvision-textrecognition-afifa.apk" to your android phone.

The development is using Java Programming Language and using Android Studio as the development tool.

If you want to run the code, you can import this "Android - Implementation" folder in Android Studio.

If you want to see the main code (java file), you can open folder below :

... \app\src\main\java\com\vhiefa\robotvision

There are four java files : Blob.java, CharSegment.java, MainActivity.java, and NeuralNetworkModel.java

The main file is MainActivity.java
